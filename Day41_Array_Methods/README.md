# Day41 - Array Methods

---

# 🇺🇸 English

## Learning Objectives

- Understand the difference between `for` loops and array methods.
- Learn how to use `map()`, `filter()`, `find()`, `some()`, and `every()`.
- Practice transforming and filtering array data.
- Understand immutable object updates with the spread operator.

## Topics

- Array Methods
- map()
- filter()
- find()
- some()
- every()
- Spread Operator (...)
- Immutable Update

## Practice

- Print user names
- Convert names to uppercase
- Filter active users
- Find a specific user
- Check conditions with `some()` and `every()`
- Add new properties using the spread operator

## Skills

- JavaScript ES6+
- Array Manipulation
- Functional Programming Basics
- Immutable Update

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)

## Future Improvements

- Learn object destructuring
- Practice complex array transformations
- Apply array methods in React projects
