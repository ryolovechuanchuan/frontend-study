📖 プロジェクト概要

Vanilla JavaScript を使用して作成した社員管理アプリです。

基本的な CRUD（Create・Read・Update・Delete）機能を実装し、社員情報を追加・編集・更新・削除できます。

DOM 操作や Array・Object を活用し、データを動的に画面へ表示します。

🔄 CRUD の流れ
Create

Employee Object を作成し、push() で配列へ追加します。

Read

map() と join() を使用して社員一覧を表示します。

Update

既存データを入力欄へ戻し、編集後に配列を更新します。

Delete

splice() を利用して社員データを削除し、画面を再描画します。
