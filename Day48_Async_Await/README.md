# Day48 - Async / Await & Fetch API

## 🇹🇼 中文

### 📖 學習重點

今天學習了 JavaScript 的 `async / await`，並將 Promise 改寫成更容易閱讀的同步流程。同時開始使用真正的 Fetch API，取得遠端資料並更新網頁畫面。

### 📚 學習內容

- async function
- await
- Promise → Async/Await
- Promise Chain → Async/Await
- try...catch
- Fetch API
- Response Object
- response.json()
- response.ok
- throw new Error()
- JSON Data Parsing
- DOM Manipulation
- Event Listener
- Random User API

### 💻 練習內容

- 使用 async / await 重寫 Promise
- 使用 try...catch 處理錯誤
- 使用 fetch() 取得 API 資料
- 解析 JSON 回傳資料
- 顯示使用者姓名
- 顯示 Email
- 顯示國家
- 顯示使用者照片
- 使用 Button 重新取得使用者資料
- 使用 response.ok 判斷 API 是否成功

### 🎯 學習心得

今天真正理解了 `async / await` 並不是新的非同步技術，而是 Promise 的另一種寫法。同時也學會 Fetch API 的完整流程：取得 Response、解析 JSON、檢查 API 狀態、更新 DOM，並利用 Random User API 建立第一個真正的 API 練習。
