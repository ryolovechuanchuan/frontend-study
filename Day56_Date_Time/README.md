# 🇹🇼 中文

## 📖 專案介紹

這是 **180 天前端工程師學習計畫 Day56** 的作品。

本專案使用 JavaScript 的 **Date 物件** 製作一個即時數位時鐘（Digital Clock），能夠顯示目前日期、時間，並根據不同時段顯示對應的問候語。

透過本專案，我學習了如何取得日期與時間資訊、格式化時間，以及利用 `setInterval()` 每秒更新畫面。

---

## 🚀 功能

- 即時日期顯示
- 即時時間顯示
- 每秒自動更新
- 星期顯示
- Good Morning / Afternoon / Evening 問候語
- 日期補零（07、08）
- 時間補零（09:05:03）
- Responsive Layout
