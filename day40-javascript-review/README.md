## ✨ Features

- Add a new todo
- Edit existing todos
- Delete todos
- Mark todos as completed
- Filter todos (All / Active / Completed)
- Clear completed todos
- Save data with Local Storage
- Responsive Design (RWD)

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage
