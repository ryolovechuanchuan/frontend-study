# 🇹🇼 中文

## 專案介紹

這是一個使用 Vanilla JavaScript 開發的 Todo List。

使用者可以新增、編輯、刪除待辦事項，並透過 Local Storage 保存資料，即使重新整理網頁也不會遺失。

此專案主要練習 JavaScript DOM 操作、事件委派（Event Delegation）、陣列方法（map、filter、find）、Local Storage 以及 Flexbox 排版。

---

## ✨ Features

- Add a new todo
- Edit existing todos
- Delete todos
- Mark todos as completed
- Filter todos (All / Active / Completed)
- Clear completed todos
- Save data with Local Storage
- Responsive Design (RWD)

This is a Todo List application built with Vanilla JavaScript.

# 🇯🇵 日本語

## プロジェクト概要

Vanilla JavaScript を使用して作成した Todo List アプリです。

タスクの追加・編集・削除・完了切り替えができ、Local Storage を利用してデータを保存するため、ページを更新しても内容は保持されます。

このプロジェクトでは DOM 操作、イベントデリゲーション、配列メソッド、Local Storage、Flexbox、レスポンシブデザインを学習しました。

---

## 主な機能

- Todo の追加
- Todo の編集
- Todo の削除
- 完了 / 未完了の切り替え
- フィルター（すべて・未完了・完了）
- 完了済み Todo の一括削除
- Local Storage 保存
- レスポンシブ対応
