# 🌐 API Playground

A web application built with **HTML, CSS, and JavaScript** that demonstrates how to work with multiple public APIs using `fetch()` and `async/await`.

This project integrates several APIs into one application and focuses on asynchronous programming, API handling, error management, and responsive UI.
